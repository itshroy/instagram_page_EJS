## Instagram Profile Clone

A simple Instagram-inspired profile page built using **Node.js**, **Express.js**, and **EJS**.

## Features

- Display Instagram-like user profile
- Dynamic rendering using EJS
- Shows username, followers, and following count
- Displays multiple posts
- Shows likes and comments for each post
- Static image serving using Express

## Tech Stack

- Node.js
- Express.js
- EJS
- HTML5
- CSS3
- JavaScript
